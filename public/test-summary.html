🔍 获取 Notion 页面 ID

使用步骤:

  1. 在新标签页打开你的博客文章页面
  2. 点击下面的按钮
  3. 复制页面 ID
  4. 在终端运行测试命令

手动查找方法:

从 Notion 数据库获取:

  1. 打开你的 Notion 数据库
  2. 找到对应的文章页面
  3. 复制页面 URL 中的 ID
  4. 格式: https://www.notion.so/your-workspace/Title-abc123...
  5. 复制 abc123... 部分（32个字符）

测试命令:

node scripts/test-summary-update.js <pageId>

例如:

node scripts/test-summary-update.js abc123def456789012345678901234ab
